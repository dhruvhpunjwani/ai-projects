# Road Defect Detection System

Computer vision project using PyTorch and OpenCV to detect road defects such as potholes and cracks.

## Features

- Deep learning classification model (ResNet18)
- Real-time detection using webcam
- Data augmentation for robustness
- Binary classification: normal vs defect
